# You are trying to contribute to Project Nomad but are unsure about the code review process and how to submit a pull request.

_Contributing to Project Nomad by submitting a pull request and following the code review process_

## Steps

1. Fork the Project Nomad repository using the GitHub web interface
2. Clone the forked repository using `git clone https://github.com/your-username/project-nomad.git`
3. Make the necessary changes and commit them using `git add .` and `git commit -m 'commit message'`
4. Push the changes to the forked repository using `git push origin feature/new-feature`
5. Submit a pull request to the main Project Nomad repository using the GitHub web interface

## Pitfalls

- ⚠️ Not following the code review process, which can lead to rejected pull requests
