# You want to integrate Project Nomad with other tools and services but are unsure about the available APIs and integration points.

_Integrating Project Nomad with other tools and services using available APIs and integration points_

## Steps

1. Use the `axios` library to make HTTP requests to external APIs
2. Use the `ws` library to establish WebSocket connections to external services
3. Use the `graphql` library to query external GraphQL APIs
4. Use the `mqtt` library to establish MQTT connections to external services
5. Use the `docker` library to integrate with Docker containers and services

## Pitfalls

- ⚠️ Not handling errors and exceptions properly, which can lead to integration issues
